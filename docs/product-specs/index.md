# product-specs 索引

## 用途

存放产品规格文档。记录功能需求、非功能需求、用户场景和验收标准。

## 当前文档

暂无独立产品规格文档。产品视角的完整描述记录在：

- `docs/PRODUCT_SENSE.md` — 产品定位、核心价值、使用场景
- `docs/FRONTEND.md` — API 端点说明
- `CLAUDE.md` — 完整功能描述（3 条翻译路径、模型覆盖链、部署目标）

若需深入某个翻译路径的规格，建议创建对应规格文档前先参考上述来源。

## 交叉引用

- 执行计划（实现追踪）：`docs/exec-plans/`
- 设计文档：`docs/design-docs/`
- 项目级产品文档：`docs/PRODUCT_SENSE.md`
